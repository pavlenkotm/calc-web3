# Snake Calculator

Простой Solidity-контракт, который объединяет классическую змейку и калькулятор. Для каждого адреса
хранится собственное состояние поля, а встроенный калькулятор может приносить бонусные очки.

## Основные идеи

- **Игра**. `startGame(width, height)` создаёт поле от 4×4 до 32×32 клеток и спавнит змею длиной 3
  клетки в центре. `move(direction)` двигает змею, проверяет столкновения и обработку яблок.
- **Калькулятор**. `calculate(left, right, operation)` работает как обычный калькулятор. Версия
  `calculateAndBoost(...)` дополнительно начисляет от 1 до 5 очков активной игре, если результат не 0.
- **Прозрачность**. Состояние можно читать через `getGameMeta(player)` и `getBody(player)`.

## Как использовать

1. Скомпилируйте контракт любой любимой тулчейн-средой (Foundry/Hardhat/Remix).
2. Задеплойте `contracts/SnakeCalculator.sol` в подходящую сеть.
3. Вызовы:
   - `startGame(12, 12)` → новая игра.
   - `move(Direction.Right)` → шаг вперёд.
   - `calculateAndBoost(2, 3, Operation.Multiply)` → умножение и бонус.

## Безопасность и ограничения

- Случайность появления яблока псевдо-рандомная и подходит лишь для демонстрационных целей.
- Доска и тело змейки хранятся целиком в цепочке, поэтому газ растёт с длиной змейки.
